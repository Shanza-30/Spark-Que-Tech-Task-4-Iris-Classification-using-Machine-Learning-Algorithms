# 🌸 Iris Classification using Machine Learning

## 📌 Project Overview

This project is based on the famous Iris dataset. The goal is to train multiple machine learning algorithms, compare their performance, and implement Spark/QueTech Task 4 for scalable data processing."
We implemented:
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

## 🎯 Objective
To understand how different machine learning algorithms perform on the same dataset and compare their accuracy.

## 📊 Dataset
We used the Iris dataset which contains:
- 150 samples
- 4 features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - 3 classes of flowers (Setosa, Versicolor, Virginica)

## 🛠️ Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## ⚙️ Steps Performed
1. Imported libraries
2. Loaded dataset
3. Created DataFrame
4. Data analysis & visualization
5. Train-test split
6. Model training:
   - KNN
   - Decision Tree
   - Random Forest
7. Model evaluation using accuracy
8. Compared results using graph
9. Confusion matrix visualization

## 📈 Results
The models were compared based on accuracy.  
Random Forest performed best in most cases.

## 📌 Conclusion
Different algorithms give different results on the same dataset.  
Random Forest generally provides better accuracy due to ensemble learning.
